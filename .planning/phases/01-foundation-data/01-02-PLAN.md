2. Create `src/components/BubbleCanvas.tsx`:
   - Use `useRef<HTMLCanvasElement>` and `useEffect` for canvas setup
   - HiDPI scaling pattern (per RESEARCH.md):
     - Get `devicePixelRatio` (or default to 1)
     - Set `canvas.width = parentWidth * dpr`, `canvas.height = parentHeight * dpr`
     - Set `canvas.style.width = parentWidth + 'px'`, `canvas.style.height = parentHeight + 'px'`
     - Call `ctx.setTransform(dpr, 0, 0, dpr, 0, 0)` (use setTransform, not scale, to avoid accumulating transforms on resize)
   - Draw dark background: `ctx.fillStyle = '#1a1a1a'; ctx.fillRect(0, 0, w, h)`
   - Draw ONE test bubble centered on the canvas:
     - Blue base color: `hsl(210, 60%, 35%)` (VN positive convention per user decision)
     - 3D sphere radial gradient overlay (adapted from reference script.js):
       ```
       createRadialGradient(cx - r*0.18, cy - r*0.22, r*0.05, cx, cy, r)
       stops: 0 -> rgba(255,255,255,0.14), 0.3 -> rgba(255,255,255,0.05),
              0.6 -> rgba(255,255,255,0), 1 -> rgba(0,0,0,0.22)
       ```
     - Glow border: `ctx.shadowBlur = 20; ctx.shadowColor = 'rgba(50, 130, 255, 0.4)'` for blue glow effect
     - Ticker label "VNM" centered in white bold text, "+2.5%" below in slightly smaller, semi-transparent white
   - Radius of test bubble: ~60px
   - Listen for window resize (or use ResizeObserver on parent) -- redraw immediately on resize (no debounce on the redraw itself, to avoid white flash)
   - Clean up listener on unmount
   - Wrap the canvas in a `<div className="relative flex-1 overflow-hidden">` so it fills remaining space below header

3. Create `src/components/App.tsx`:

   - First child: `<Header />`
   - Second child: `<BubbleCanvas />` (flex-1 to fill remaining space)
   - No other elements

4. Update `src/main.tsx`:

   - Keep the `import './index.css'` and ReactDOM.createRoot render call

5. Remove any remaining Vite boilerplate (App.css, logo.svg, etc.) that was not already cleaned up in Plan 01.
